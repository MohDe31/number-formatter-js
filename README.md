## Usage

```js
formatFromMask(1000000, "$0,000.00"); // "$1,000,000.00"
formatFromMask(1000000, "$+0,000.00"); // "$+1,000,000.00"
formatFromMask(-1000000, "$+0,000.00"); // "$-1,000,000.00"
formatFromMask(1000000, "+++-> +0,000.00USD"); // "+++-> +1,000,000.00USD"
```
